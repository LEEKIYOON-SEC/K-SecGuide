# K-SecGuide

주요정보통신기반시설 취약점 분석·평가 가이드

## 소개

한국인터넷진흥원(KISA)에서 발간한 주요정보통신기반시설 취약점 분석·평가 가이드(관리/물리/기술)를 웹에서 쉽게 검색하고 참조할 수 있도록 구성한 사이트입니다.

- 관리적 분야: 118개 항목 (A-1 ~ A-118)
- 물리적 분야: 18개 항목 (P-1 ~ P-18)
- 기술적 분야: 363개 항목 (11개 플랫폼)
- 관계 법규: 14개 법령 매핑
- 합계: 499개 점검항목
