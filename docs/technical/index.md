# 기술적 분야

주요정보통신기반시설 **기술적 취약점 분석·평가** 점검항목입니다.

현재 **6개 플랫폼**의 점검항목이 수록되어 있으며, 나머지 플랫폼은 순차적으로 추가됩니다.

| 플랫폼 | 코드 | 항목 수 | 상태 |
|:-------|:-----|:--------|:-----|
| UNIX 서버 | U-01 ~ U-67 | 67개 | 수록 완료 |
| Windows 서버 | W-01 ~ W-64 | 64개 | 수록 완료 |
| 웹 서비스(보안설정) | WEB-01 ~ WEB-26 | 26개 | 수록 완료 |
| 보안 장비 | S-01 ~ S-23 | 23개 | 준비 중 |
| 네트워크 장비 | N-01 ~ N-38 | 38개 | 수록 완료 |
| 제어시스템 | C-01 ~ C-51 | 51개 | 준비 중 |
| PC | PC-01 ~ PC-18 | 18개 | 준비 중 |
| DBMS | D-01 ~ D-26 | 26개 | 준비 중 |
| 이동통신 | M-01 ~ M-04 | 4개 | 준비 중 |
| 웹(Web App) | 약어코드 21종 | 21개 | 수록 중 |
| 가상화 장비 | HV-01 ~ HV-25 | 25개 | 수록 완료 |
